SeaSense — Version History
==========================


v8.8.2 — 2026-04-23
---------------------
Light theme visual polish:
- Main window background set to #fbefdc (warm off-white)
- Hero/header background now matches the sidebar (--surface, #dce8fb)
- App name and tagline text use deep blue (--text, #0c306a) instead of white
- Column layout alternating row second colour updated to #c5d9f8
- Tides strip and water temperature text set to secondary blue (#1e3a6e) in light mode across all layouts


v8.7.4 — 2026-04-23
---------------------
- Added service worker (sw.js) with network-first strategy: every app open fetches the latest version from GitHub Pages, with offline cache fallback
- Service worker registered in index.html; cache name is version-keyed so each release automatically evicts stale caches on all devices


v8.7.3 — 2026-04-23
---------------------
- Wizard result window is now 30% narrower (max-width 392px) while the question form stays at full width (560px)
- Smooth width transition (0.25s ease) between the form and result states
- Width resets correctly when navigating back to the form or closing the wizard


v8.0.0 — 2026-04-10
---------------------
- Hourly forecast expansion: click the "+Nh" button between any two 3-hour cards (Hourly Box layout) to reveal intermediate hourly slots
- Column layout: click the ▾ chevron in any time cell to expand the intermediate hourly rows inline
- Intermediate slots are visually distinct (dimmed, thin status bar) so the 3-hour anchors remain the primary view
- All models now fetch full hourly data; night slots (21:00, 00:00, 03:00) remain hidden


v3.7.7 — 2026-04-10
---------------------
- Fixed wind and wave arrow direction: arrows now point TO the direction the wind/waves are heading (not FROM)
- Added Wind.png icon asset


v3.7.6 — 2026-04-09
---------------------
- Removed ICON-LAM IL wind model
- Fixed status bar not clearing between fetches
- Auto-refresh now triggers immediately when the wind model is changed


v3.7.4 — 2026-04-07
---------------------
- Added ICON-LAM IL wind model: Israel Meteorological Service 2.5 km resolution model
  fetched directly from ims.gov.il; automatic CORS-proxy fallback for local file usage;
  5-day hourly forecast; nearest-location matching for all preset and custom coordinates

v1.0.0 — 2026-03-31
---------------------
Initial release.
- Marine weather forecast for Israeli coastal locations
- GO / CAUTION / NO-GO status per 3-hour slot based on wind, gust, and wave thresholds
- Customisable thresholds (GO / CAUTION / NO-GO)
- Wind unit toggle: knots / km/h
- Multiple forecast models via Open-Meteo (no API key required)
- Model descriptions and reordered model list in sidebar
- Rain probability on cards and day pills
- Location selector with preset Israeli coastal spots
- Dark nautical theme
